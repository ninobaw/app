# 💬 Guide d'Utilisation - Dialogue Conversationnel

## 🎯 **Vue d'ensemble**

Le dialogue conversationnel remplace l'ancienne interface complexe par une approche moderne de type "chat" pour gérer les propositions de réponse aux correspondances.

## 👥 **Utilisateurs concernés**

- **Directeurs** (DIRECTEUR, SOUS_DIRECTEUR)
- **Directeur Général** (DIRECTEUR_GENERAL)

## 🚀 **Comment accéder au dialogue**

### **Pour les Directeurs :**

1. **Connectez-vous** avec un compte directeur
   ```
   Exemples de comptes :
   - ahmed.benali@tav.aero / password123
   - fatma.trabelsi@tav.aero / password123
   ```

2. **Accédez au Dashboard Directeur**
   ```
   URL: /director-dashboard
   ```

3. **Trouvez la section "Mes Propositions de Réponse"**
   - Vous verrez les correspondances qui vous sont assignées
   - Chaque correspondance a un bouton "Voir détails"

4. **Cliquez sur "Voir détails"**
   - Le dialogue conversationnel s'ouvre
   - Interface moderne de type chat

### **Pour le Directeur Général :**

1. **Connectez-vous** avec le compte DG
   ```
   Email: mohamed.sassi@tav.aero
   Mot de passe: password123
   ```

2. **Accédez au Dashboard DG**
3. **Cliquez sur "Ouvrir conversation"** sur une correspondance avec propositions

## 🎨 **Interface du dialogue conversationnel**

### **Design moderne :**
- **Zone de messages** chronologique comme WhatsApp
- **Messages colorés** selon le rôle (bleu pour directeur, violet pour DG)
- **Timestamps** sur chaque message
- **Badges d'action** (APPROVE, REQUEST_REVISION, etc.)
- **Gestion des fichiers** intégrée

### **Composants de l'interface :**

#### **Header :**
- 💬 Icône de conversation
- Titre : "Conversation - Proposition de Réponse"
- Sujet de la correspondance

#### **Zone principale :**
- **Historique des messages** (chronologique)
- **Zone de saisie** avec placeholder
- **Zone drag & drop** pour fichiers
- **Boutons d'action** selon votre rôle

#### **Zone de saisie :**
- **Textarea** responsive pour rédiger
- **Upload de fichiers** par drag & drop ou clic
- **Bouton d'envoi** contextuel

## 🔄 **Workflow d'utilisation**

### **1. Directeur crée une proposition :**
```
1. Ouvrir le dialogue sur une correspondance assignée
2. Rédiger la proposition dans la zone de texte
3. Ajouter des fichiers si nécessaire (drag & drop)
4. Cliquer sur "Envoyer proposition"
5. Le message apparaît dans l'historique (encadré bleu)
```

### **2. DG donne un feedback :**
```
1. Voir la proposition du directeur dans l'historique
2. Choisir "Demander révision" ou "Approuver"
3. Rédiger les commentaires et consignes
4. Envoyer le feedback
5. Le message apparaît dans l'historique (encadré violet)
```

### **3. Directeur révise (si nécessaire) :**
```
1. Lire les commentaires du DG dans l'historique
2. Modifier la proposition selon les consignes
3. Ajouter des documents mis à jour
4. Cliquer sur "Envoyer révision"
5. Le cycle continue jusqu'à l'approbation
```

## 📎 **Gestion des fichiers**

### **Upload :**
- **Drag & drop** : Glissez vos fichiers dans la zone
- **Clic** : Cliquez sur la zone pour sélectionner
- **Types supportés** : PDF, DOC, DOCX, images

### **Affichage :**
- **Nom du fichier** avec icône
- **Taille** du fichier
- **Bouton de téléchargement** pour chaque fichier

## 🎯 **Actions disponibles**

### **Pour les Directeurs :**
- **Créer proposition** : Première proposition de réponse
- **Réviser proposition** : Modifier selon les consignes du DG
- **Ajouter documents** : Joindre des fichiers

### **Pour le Directeur Général :**
- **Demander révision** : Donner des consignes de modification
- **Approuver** : Valider la proposition
- **Ajouter commentaires** : Préciser les attentes

## 🎨 **Codes couleur**

### **Messages :**
- **Bleu** : Messages des directeurs (propositions, révisions)
- **Violet** : Messages du DG (feedbacks, approbations)
- **Vert** : Messages d'approbation finale
- **Jaune** : Messages de révision

### **Badges d'action :**
- **APPROVE** : Vert
- **REQUEST_REVISION** : Orange
- **REVISED** : Bleu
- **DRAFT** : Gris

## 📱 **Responsive Design**

L'interface s'adapte à tous les écrans :
- **Desktop** : Interface complète avec sidebar
- **Tablet** : Interface adaptée
- **Mobile** : Interface optimisée pour mobile

## 🔍 **Historique complet**

### **Tout est conservé :**
- **Tous les messages** avec timestamps
- **Toutes les actions** avec badges
- **Tous les fichiers** avec liens de téléchargement
- **Toute la chronologie** des échanges

### **Navigation :**
- **Scroll automatique** vers le dernier message
- **Recherche** dans l'historique (future fonctionnalité)
- **Export** de la conversation (future fonctionnalité)
